Para gerar um DrivingPlaybackFile rode o programa driving_playback_build enquanto faz playback de um log.

>> ./driving_playback_build -file driving_playback_file.txt

Para executar um DrivingPlaybackFile rode o programa driving_playback_play.

>> ./driving playback_play
